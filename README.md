# Fans CRM test task

## Info

* Framework: **NestJS**
* Node version: **20 lts**

## Launch

* Direct:
> Prepare:
    ```bash
      npm install -g @nestjs/cli & npm i
    ``` <hr/>
> Run:
    ```bash
        npm run start
    ``` <hr/>
> Build:
    ```bash
        npm run build
    ```

* Docker:
> Run
    ```bash 
      docker-compose up -d --build
    ``` <hr/>
> Close 
    ```bash
    docker-compose down --rmi all -v
    ```
