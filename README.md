# Fans CRM test task

## Info

* Framework: **NestJS**
* Node version: **20 lts**

## Launch

* Direct:
> Prepare:
    ```
        npm install -g @nestjs/cli & npm i
    ``` <hr/>
> Run:
    ```
        npm run start
    ``` <hr/>
> Build:
    ```
        npm run build
    ```

* Docker:
> Run:
    ```
        docker-compose up -d --build
    ``` <hr/>
> Close: 
    ```
        docker-compose down --rmi all -v
    ```
